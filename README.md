# Sistema Integral de Gestión de Clientes, Servicios y Reservas

## Fase 4 - Programación

Proyecto desarrollado para el curso de Programación de la Universidad Nacional Abierta y a Distancia - UNAD.

## Descripción

Este proyecto implementa un sistema orientado a objetos para la empresa Software FJ, permitiendo gestionar clientes, servicios y reservas sin utilizar bases de datos. La información se maneja mediante objetos y listas internas, y los eventos importantes se registran en un archivo de logs.

## Funcionalidades principales

- Registro de clientes.
- Validación de datos de clientes.
- Creación de servicios.
- Reserva de salas.
- Alquiler de equipos.
- Asesorías especializadas.
- Confirmación y cancelación de reservas.
- Procesamiento de pagos.
- Manejo de errores mediante excepciones personalizadas.
- Registro de eventos y errores en el archivo `eventos.log`.
- Simulación de operaciones válidas e inválidas.

## Conceptos aplicados

- Programación orientada a objetos.
- Abstracción.
- Herencia.
- Polimorfismo.
- Encapsulamiento.
- Clases abstractas.
- Excepciones personalizadas.
- Bloques `try/except`, `else` y `finally`.
- Logging.
- Encadenamiento de excepciones.
